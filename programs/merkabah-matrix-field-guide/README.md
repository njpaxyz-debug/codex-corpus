# Merkabah Matrix Field Guide

A pocket-sized interactive textbook application that combines:

- Merkavah / merkaba orientation notes
- Star-tetrahedron and stella-octangula geometry
- OCTANGULA-8 field-mapper calculations
- Cardinal-direction correspondence research
- Aether / Center crosswalks
- Gematria exploration
- Searchable correspondence matrix rows
- Relationship-map edges
- Quiz and prototype build sheet

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
npm run preview
```

## Notes

The app treats historical merkavah, modern merkaba sacred geometry, empirical geometry, and esoteric correspondence as distinct layers. Mathematical and sensor-style claims power the calculator and OCTANGULA-8 tools; symbolic correspondences are organized as searchable research records rather than scientific assertions.
