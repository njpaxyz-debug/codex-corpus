## Codex anatomy

Every Codex entry uses this structure:

1. **Designation** — three-digit permanent entry number + title.
2. **Domain Tags** — engine tags such as `TBFC`, `MM`, `ARG`, `RPG`, `SCI`.
3. **Core Thesis** — crystallized insight of the entry.
4. **Biological Ground** — anatomical, physiological, or biophysical reference material.
5. **Technological Mirror** — engineering, signal-processing, computational, or materials-science analogue.
6. **Practice Gate** — embodied action, somatic exercise, frequency prescription, meditation, or movement sequence.
7. **Cross-References** — related Codex entries.
8. **Source Stratum** — key references and source layer.
