# Codespaces Loading Guide

From the repo root:

```bash
cd apps/codex-astrolabe
npm install
npm run verify
npm run dev
```

Open the forwarded Vite port. If `vite: not found`, run `npm install` again from `apps/codex-astrolabe`. If the preview does not open, restart `npm run dev` and check the Codespaces **Ports** panel.
